# Flowly

Flowly is a polished SaaS landing page for an AI workflow automation product, built as a portfolio project with Next.js 15, TypeScript, Tailwind CSS, Framer Motion, React Hook Form, and Zod.

Live site: <https://flowly.amjadyahia.com/>

## Stack

- Next.js 15 App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- React Hook Form
- Zod
- pnpm

## Features

- Sticky animated navbar with scroll state
- Motion-driven hero section with product mockup
- Trusted-by logos bar
- Responsive feature grid with in-view animations
- Three-step how-it-works section
- Interactive pricing toggle
- Animated FAQ accordion
- Waitlist form with validation, loading, and success states
- Open Graph metadata and social preview image

## Getting Started

```bash
pnpm install
pnpm dev
```

Open `http://localhost:3000`.

## Scripts

```bash
pnpm dev
pnpm build
pnpm start
pnpm typecheck
```
